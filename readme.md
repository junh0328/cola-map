# 📜 주요 기능

로그인 이전 단계와 로그인 이후 단계로 나눠집니다

## 로그인 이전

메인 페이지

- 주변 가게(음식점)의 좌표를 통해 가게 정보 얻기
- 가게에 제보된 정보 얻기 (어떤 음료수를 파는지?)
- 키워드로 검색하기

카테고리 페이지

- 펩시 제보된 가게 전체 보기
- 코카콜라 제보된 가게 전체 보기

## 로그인

- 카카오로 로그인 (소셜 로그인)
- 로컬 스토리지 이용 (JWT 기반)

## 로그인 이후

메인 페이지

- 가게 음료 제보하기 (카테고리 및 글)
- 본인 글 수정하기 / 삭제하기
- 가게 신고하기 (삭제요청)
- 로그아웃 하기

마이 페이지 이용 가능

- 내가 한 제보 보기 (목록형/카드형)
- 닉네임 변경
- 로그아웃
- 만든 사람들 소개
